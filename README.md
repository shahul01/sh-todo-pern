# sh-todo-pern
Todo app created with PERN stack with authentication

## tech stacks - PERN
  - frontend - React with Next
  - backend - NodeJS + Express + Postgress

## to run

  - frontend
  ```bash

    cd frontend

    # install
    npm i

    # run
    npm run dev
  ```

  - backend
  ```bash

    cd backend

    # install
    npm i

    # run
    npm run dev
  ```
