# sh-todo-pern
Todo app created with PERN stack with authentication

## tech stacks - PERN
  - frontend - React with Next
  - backend - NodeJS + Express + Postgress

## to run

  - frontend
  ```bash
    # install
    npm i
    # or its alternatives like yarn

    # run
    npm run dev
    # or
    yarn dev
  ```

  - backend - follow its package json
